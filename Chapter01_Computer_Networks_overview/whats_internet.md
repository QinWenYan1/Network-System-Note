# 1.1 什么是 Internet？

> 📚 **来源**：中科大郑烇老师《计算机网络》第1章 - 1.1 什么是Internet

---

## 一、从具体构成角度理解 Internet

### 1.1 基本组成要素

Internet 由以下三部分构成：

| 要素 | 说明 | 示例 |
|------|------|------|
| **节点** | 主机 + 网络交换设备 | PC、手机、服务器、路由器、交换机 |
| **边** | 通信链路 | 光纤、同轴电缆、无线电、卫星 |
| **协议** | 控制发送和接收消息的规则 | TCP、IP、HTTP、FTP、PPP |

### 1.2 关键概念

- **主机（Host）= 端系统（End System）**
  - 运行网络应用程序
  - 包括：PC、服务器、智能手机、物联网设备等

- **通信链路（Communication Links）**
  - **接入网链路**：主机连接到互联网的链路
  - **主干链路**：路由器之间的链路
  - **传输速率** = **带宽**（单位：bps）

- **分组交换设备**
  - **路由器（Router）**：转发分组（packets）
  - **交换机（Switch）**：链路层交换

### 1.3 Internet 的本质

> 🌐 **Internet = "网络的网络"（Network of Networks）**

- 松散的层次结构，互连的 ISP
- **公共 Internet** vs **专用 Intranet**

### 1.4 Internet 标准

| 标准 | 说明 |
|------|------|
| **RFC** | Request for Comments（请求评论） |
| **IETF** | Internet Engineering Task Force（互联网工程任务组） |

## 二、从服务角度理解 Internet

### 2.1 分布式应用

Internet 支持各种**使用通信设施进行通信的分布式应用**：

- Web 浏览
- VoIP（网络电话）
- Email
- 分布式游戏
- 电子商务
- 社交网络

### 2.2 通信基础设施

为应用程序提供**编程接口（通信服务）**：

| 服务类型 | 特点 | 协议 |
|----------|------|------|
| **无连接不可靠服务** | 不保证到达、不保证顺序 | UDP |
| **面向连接的可靠服务** | 保证到达、按序传输 | TCP |

> 💡 **类比**：类似于邮政服务，你可以选择普通邮件（不可靠）或挂号信（可靠）

---

## 三、什么是协议？

### 3.1 定义

> 📋 **协议（Protocol）** 定义了在两个或多个通信实体之间交换的：
> - 报文**格式**
> - 报文**次序**
> - 传输/接收时采取的**动作**

### 3.2 人类协议 vs 网络协议

| 人类协议 | 网络协议 |
|----------|----------|
| "几点了？" | TCP 连接请求 |
| "2:00" | TCP 连接响应 |
| "我有个问题" | GET `http://www.example.com` |
| ... | <文件> |

### 3.3 协议的核心特征

- ✅ 机器之间的协议（而非人与人之间的协议）
- ✅ Internet 中**所有通信行为**都受协议制约
- ✅ 发送特定的消息
- ✅ 收到消息时采取特定行动

## 五、重点总结

### 5.1 Internet 的两种视角

| 视角 | 关注点 | 关键词 |
|------|--------|--------|
| **构成角度** | 硬件设备、连接方式 | 主机、链路、路由器、协议 |
| **服务角度** | 为应用提供的通信能力 | 分布式应用、API、可靠/不可靠服务 |

### 5.2 核心概念对比

```
主机（端系统） ──────► 运行应用程序，位于网络边缘
       │
       ├── 通过接入网链路 ──► 连接到 Internet
       │
       └── 通过主干链路 ────► 路由器之间互联

路由器/交换机 ──────► 转发分组，构成网络核心
       │
       └── 遵循协议进行数据交换
```

### 5.3 关键术语速记

| 术语 | 英文 | 含义 |
|------|------|------|
| 端系统 | End System / Host | 运行应用的设备 |
| 分组 | Packet | 网络中传输的数据单位 |
| 带宽 | Bandwidth | 链路的传输速率（bps） |
| 协议 | Protocol | 通信规则和约定 |
| ISP | Internet Service Provider | 互联网服务提供商 |

---

> ⚡ **一句话总结**：Internet 是一个由数以亿计的互联计算设备组成的网络，通过通信链路和分组交换设备连接，遵循协议进行数据传输，为分布式应用提供通信服务的基础设施。
